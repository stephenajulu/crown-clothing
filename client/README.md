> A ReactJS x Firebase ecommerce clothing store

## Technologies used
React DOM, Router, Redux (sagas & thunks), Hooks, Styled Components, Stripe and Firebase Firestore.


## Install

```sh
yarn
```

## Usage

```sh
yarn start
```

## Run tests

```sh
yarn run test
```
